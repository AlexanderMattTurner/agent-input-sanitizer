- ANSI escape stripping no longer backtracks quadratically on adversarial input (`ESC` followed by many `;#` with no terminator), closing a polynomial-ReDoS denial-of-service vector in Layer 1.
- `urlHost` now classifies relative vs. absolute URLs by parsing rather than a substring prefix test, so an off-origin host that merely starts with the relative-resolution sentinel (`relative.invalid.evil.example`) is labeled by its real host instead of as a relative URL.
